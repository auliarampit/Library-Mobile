## Description
An application for online libraries, in the application there are several features, such as loans, donations, and so forth

## Clone or download
`$ git clone https://github.com/auliarampit/Library-Mobile.git`
`$ cd Library-Mobile`
`$ npm install`

## Start Development
`$ react-native run-android`
